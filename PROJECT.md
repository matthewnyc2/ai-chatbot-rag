## Portfolio Story
A customer-facing AI chat assistant that answers questions about a company's documentation using RAG (Retrieval-Augmented Generation). Upload PDFs/docs, the system chunks and embeds them, and the chatbot answers questions grounded in those documents.

## Key Features to Showcase
- Document upload and chunking pipeline
- Vector embedding and semantic search
- Multi-turn conversational UI
- Source citation in responses
- Configurable LLM provider (OpenAI, Anthropic, Ollama for local)

## Standalone
Yes — runs with Ollama for $0 local LLM inference
