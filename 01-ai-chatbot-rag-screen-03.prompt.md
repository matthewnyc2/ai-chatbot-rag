# RAG Chatbot - Screen 03: Academic Research Hub (Lexis AI)

## Overall Vibe
Academic, professional, research-oriented. Emphasizes knowledge synthesis and intelligent inquiry. Scholarly yet modern interface for academic and research use cases.

## Design System Block
- **Color**: Deep purple/indigo theme (#0f131c background, #6d5bff primary, #c6c0ff accent). Gradient accents. High contrast light text.
- **Typography**: Manrope headlines, Inter body, Space Grotesk labels. Sizes: 60px display, 20px headlines, 16px body.
- **Spacing**: 24px margins, 16px padding. Generous whitespace. Asymmetric three-column grid.
- **Components**: 4px-6px radius, smooth gradients, glassmorphic panels, primary gradient buttons.

## Page Structure - Research Hub Layout

### Top Navigation (Full Width)
- "Lexis AI" branding on left
- Navigation tabs: Research (active), Library, Models, Settings
- Right side: Notifications, Help, Profile avatar

### Left Sidebar (256px)
- "Research Hub" title with "Academic Futurist Engine" subtitle
- "New Research Thread" button (gradient)
- Main Navigation with sections: Current Session (active/highlighted), History, Archives, Collections
- Bottom: Model Config, Support
- Material Design icons for each nav item

### Center Content Area
- Welcome hero: "Hello, Researcher" headline
- Description: "Synthesize insights from your private knowledge base..."
- Four suggested query cards in 2x2 grid: Analyze, Summarize, Evaluate, Draft
- Icons and brief descriptions for each query type
- Fixed bottom: Input composer with paperclip icon, text input, gradient send button

### Right Sidebar (320px)
- "Knowledge Base" header with search input (search icon left-aligned)
- Indexing Status: Progress bar showing 14/15 documents ready (93% complete)
- Document List: 4 documents shown with:
  - File type icons (PDF, DOCX)
  - Filename (truncated)
  - File size and date
  - Hover effects for interactivity
- "View All Sources" button at bottom
